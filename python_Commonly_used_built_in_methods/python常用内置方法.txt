# 基于  https://docs.python.org/3/   python文档
python3中常用的内置方法
abs(): 返回一个数值的绝对值
all(): 接受一个可迭代对象作为参数，如果该可迭代对象中所有元素都为True，则返回True，否则返回False
any(): 接受一个可迭代对象作为参数，只要该可迭代对象中有一个元素为True，则返回True，否则返回False
ascii(): 返回一个字符串的ASCII表示
bin(): 将一个整数转换为二进制字符串
bool(): 将一个值转换为布尔类型
bytearray(): 创建一个可变字节数组
bytes(): 创建一个不可变字节数组
callable(): 判断一个对象是否可调用
chr(): 返回指定ASCII码对应的字符
classmethod(): 用来指示一个方法是类方法
compile(): 编译源代码为代码对象或AST对象
complex(): 创建一个复数
delattr(): 删除对象的属性
dict(): 创建一个字典
dir(): 返回一个对象的所有属性和方法
divmod(): 返回两个数的商和余数
enumerate(): 枚举一个可迭代对象
eval(): 将一个字符串作为Python表达式执行
exec(): 执行一段Python代码
filter(): 过滤一个可迭代对象，并返回一个新的可迭代对象
float(): 创建一个浮点数
format(): 格式化一个字符串
frozenset(): 创建一个冻结集合
getattr(): 获取对象的属性值
globals(): 返回全局变量的字典
hasattr(): 判断对象是否有指定的属性
hash(): 获取一个对象的哈希值
help(): 显示帮助信息
hex(): 将一个整数转换为十六进制字符串
id(): 返回对象的唯一标识符
input(): 获取用户输入
int(): 创建一个整数
isinstance(): 判断一个对象是否属于某个类
issubclass(): 判断一个类是否是另一个类的子类
iter(): 返回一个迭代器
len(): 返回一个序列的长度
list(): 创建一个列表
locals(): 返回当前作用域的局部变量的字典
map(): 对一个可迭代对象中的每个元素执行一个函数，并返回一个新的可迭代对象
max(): 返回可迭代对象中的最大值
memoryview(): 创建一个内存视图对象
min(): 返回可迭代对象中的最小值
next(): 返回迭代器的下一个元素
object(): 创建一个基本对象
oct(): 将一个整数转换为八进制字符串
open(): 打开一个文件
ord(): 返回一个字符的ASCII码值
pow(): 计算一个数的幂
print(): 输出到控制台
property(): 创建一个属性
range(): 创建一个数字范围
repr(): 返回一个对象的字符串表示形式
reversed(): 将一个序列反转，并返回一个新的序列
round(): 四舍五入一个数
set(): 创建一个集合
setattr(): 设置对象的属性值
slice(): 创建一个切片对象
sorted(): 对一个可迭代对象进行排序
